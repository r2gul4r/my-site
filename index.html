About Me

과거 e스포츠 경험을 통해 매 경기마다 쌓이는 방대한 로그와 패턴을 분석하는 데에서 프로그래밍의 매력을 처음 느꼈습니다. 단순히 소비되는 정적 데이터에 생명을 불어넣는 것에 빠져, AI와 데이터 자동화, 그리고 이를 사용자와 연결하는 풀스택 엔지니어링으로 영역을 넓혀왔습니다.

업무나 일상에서 일어나는 비효율과 반복적인 불편함을 발견하면 그냥 지나치지 않습니다. "이것을 자동화할 수 없을까?"라는 질문을 던지며, 기획한 아이디어를 빠르게 메모하고, 즉각 프로토타입으로 제작해 검증하는 실행력을 지니고 있습니다.

AI 모델 파이프라인부터 클라이언트 웹 환경까지 전체 레이어를 관통하는 견고한 아키텍처를 추구합니다.

Focus Areas

주로 다음과 같은 기술 및 실무 가치를 극대화하는 것에 집중하고 연구합니다.

01. AI Workflow Design

생성형 AI 모델을 단순 호출하는 단계를 넘어 입력 전처리, 비즈니스 규칙 판단, 검증, 최종 모니터링 화면까지 유기적으로 이어지는 AI 오케스트레이션 흐름을 설계합니다.

02. Data Collection & ETL

공개 웹 데이터의 분산 수집부터 데이터 클렌징, 구조적 정규화, 적재 파이프라인 및 비정형 문서 내 주요 메타데이터 추출 엔진을 구현합니다.

03. Full-stack Product Dev

React/Vite 및 Express, Spring Boot 기반의 견고한 시스템을 제작합니다. 컴포넌트 단위 설계와 신뢰도 높은 API 엔드포인트 구축을 핵심으로 삼습니다.

04. Local Automation Tools

개발자 및 실무자의 효율성을 위해 Electron, Node.js, PowerShell 스크립트를 활용해 OS 레벨에서 원클릭으로 작동하는 로컬 실행 자동화 환경을 구축합니다.
